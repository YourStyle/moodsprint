# Профиль пользователя

Страница `/profile` — личный кабинет с статистикой и достижениями.

## Карточка пользователя

Верхний блок профиля:
- **Аватар** — фото из Telegram или инициал
- **Имя** — имя пользователя
- **Username** — @username из Telegram

## XP и уровень

Блок прогресса:
- **Текущий уровень** — число и название
- **XP** — текущее количество очков опыта
- **Прогресс-бар** — до следующего уровня
- **XP до следующего уровня** — сколько осталось

## Серия (стрик)

Информация о серии активности:
- **Текущая серия** — дней подряд
- **Рекорд** — самая длинная серия

Серия увеличивается при любой активности в течение дня.

## Статистика

Сетка из 4 карточек:

| Метрика | Описание |
|---------|----------|
| Задачи | Количество выполненных задач |
| Подзадачи | Количество выполненных подзадач |
| Время фокуса | Общее время в часах |
| Макс. серия | Рекорд серии в днях |

## Достижения

### Разблокированные

Список полученных достижений:
- Иконка достижения
- Название
- Описание
- Дата получения
- Количество XP награды

### В процессе

Достижения с прогрессом:
- Название и описание
- Прогресс-бар
- Текущий/требуемый прогресс

Подробнее о достижениях в [05-gamification.md](./05-gamification.md)

## Выход

Кнопка "Выйти" в нижней части профиля:
- Очищает сессию
- Перенаправляет на страницу входа
- Данные сохраняются на сервере

## Данные профиля

Хранятся в UserProfile:
- **Продуктивное время** — когда пользователь предпочитает работать
- **Любимые типы задач** — категории задач
- **Личные вызовы** — с чем пользователь борется
- **Описание работы** — контекст деятельности
- **Цели** — личные цели пользователя

Эти данные используются для:
- Умной сортировки задач
- Персонализации AI-рекомендаций
- Адаптации стратегий декомпозиции
