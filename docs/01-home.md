# Главная страница

Главная страница — это центральный хаб приложения, откуда пользователь управляет своим днём.

## Приветствие

Динамическое приветствие в зависимости от времени суток:
- **Утро (до 12:00)**: "Доброе утро"
- **День (12:00-17:00)**: "Добрый день"
- **Вечер (после 17:00)**: "Добрый вечер"

Отображается имя пользователя и аватар.

## Календарь недели

Горизонтальный календарь на текущую неделю:
- Быстрый выбор дня для просмотра задач
- Выделение текущего дня
- Выделение выбранного дня

При выборе дня список задач фильтруется по этой дате.

## Список задач

Компактный список задач на выбранный день:
- Отображается до **10 задач** максимум
- Кнопка "Посмотреть все" если задач больше
- Для каждой задачи показывается:
  - Название
  - Прогресс выполнения подзадач (прогресс-бар)
  - Кнопка быстрого старта фокус-сессии

### Кнопка "Добавить"
Открывает модальное окно создания задачи с полями:
- Название (обязательно)
- Описание
- Приоритет (низкий/средний/высокий)
- Дата выполнения

## Уведомление о перенесённых задачах

Если задачи были автоматически перенесены с прошлых дней, показывается баннер:
- Количество перенесённых задач
- Список задач с повышенным приоритетом
- Кнопка закрытия

## AI-помощник

Блок с анимированным AI-блобом:
- Статус последней проверки настроения
- Кнопка "Отметить настроение" или "Обновить настроение"
- Подсказка о возможностях AI

## XP и прогресс

Компонент XP-бара показывает:
- Текущий уровень и его название
- Количество XP
- Прогресс до следующего уровня
- Анимация при получении XP

## Серия (стрик)

Бейдж с информацией о серии:
- Текущая серия дней
- Рекорд (самая длинная серия)
- Иконка огня с анимацией

## Ежедневные цели

Блок с тремя ежедневными целями:
1. **Время фокуса** — 60 минут
2. **Подзадачи** — 5 штук
3. **Проверка настроения** — 1 раз

Показывается прогресс по каждой цели и общий статус.

## Ежедневный бонус

При входе в приложение показывается модальное окно:
- Анимированная иконка подарка
- Количество XP к получению
- Множитель за серию
- Кнопка "Забрать награду"

Бонус можно получить один раз в день.

## Модальное окно настроения

При нажатии на кнопку настроения открывается форма:
- Выбор уровня настроения (1-5)
- Выбор уровня энергии (1-5)
- Опциональная заметка
- Кнопка отправки

После отправки начисляется XP и обновляется AI-стратегия.
